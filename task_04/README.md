# task_04 - Моделирование подбора ключа

## Постановка

Есть связка из `n` ключей, только один ключ подходит к двери.  
Ключи пробуются последовательно, неподошедший ключ повторно не используется.

Эксперимент может закончиться на попытке `1, 2, ..., n`.

Требуется:
- показать, что вероятность каждого исхода одинакова (`1/n`);
- реализовать серию из `K` экспериментов;
- для каждого эксперимента вывести, на какой попытке дверь открылась;
- посчитать эмпирические вероятности исходов.

## Идея модели

В каждом эксперименте:
- формируем случайный порядок из `n` ключей (равновероятная перестановка);
- номер позиции правильного ключа и есть число попыток до успеха `T`.

Тогда аналитически:

- `P(T = m) = 1/n`, `m = 1..n`.

## Аргументы программы

```text
./task_04 n K
```

- `n` - число ключей;
- `K` - число экспериментов в серии.

## Структура

- `src/main.cpp` - решение на C++17.

## Запуск в Ubuntu (WSL)

### 1) Открыть Ubuntu

Из PowerShell:

```powershell
wsl -d Ubuntu
```

### 2) Перейти в папку задания

```bash
cd /mnt/c/CursWorkTV/task_04
```

### 3) Скомпилировать

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic src/main.cpp -o task_04
```

### 4) Запустить

```bash
./task_04 5 20
```

Где:
- `5` - количество ключей;
- `20` - количество экспериментов.

## Что выводится

- список всех `K` экспериментов и результат каждого;
- для каждого исхода `T = 1..n`:
  - число попаданий;
  - эмпирическая вероятность;
  - аналитическая вероятность `1/n`;
  - модуль разности;
- проверка суммы эмпирических вероятностей.
