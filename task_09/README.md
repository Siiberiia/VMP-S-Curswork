# task_09 — Условная вероятность (монеты в кошельке)

## Условие

В кошельке `n` монет:

- у `k` монет орёл на **обеих** сторонах (всегда выпадает орёл);
- у остальных `n - k` монет орёл **только на одной** стороне (обычная монета: орёл и решка с вероятностью `1/2`).

Случайно выбирают монету и **четыре раза подряд** подбрасывают.

Нужно найти **условную вероятность**: при четвёртом подбрасывании снова орёл, **если** на трёх предыдущих тоже был орёл — и оценить её симуляцией.

## Аналитика (для отчёта)

Обозначим тип монеты: `D` — двуглая, `F` — честная.

- `P(D) = k/n`, `P(F) = (n-k)/n`.
- `P(O_1 O_2 O_3 | D) = 1`.
- `P(O_1 O_2 O_3 | F) = (1/2)^3 = 1/8`.

По формуле полной вероятности:

`P(O_1 O_2 O_3) = (k/n) * 1 + ((n-k)/n) * (1/8) = (7k + n) / (8n)`.

Аналогично для четырёх орлов подряд на честной монете: `(1/2)^4 = 1/16`:

`P(O_1 O_2 O_3 O_4) = (k/n) * 1 + ((n-k)/n) * (1/16) = (15k + n) / (16n)`.

Условная вероятность:

`P(O_4 | O_1 O_2 O_3) = P(O_1 O_2 O_3 O_4) / P(O_1 O_2 O_3) = (15k + n) / (2(7k + n))`.

Частные случаи:

- `k = 0`: только честные монеты → ответ `1/2` (четвёртый бросок независим от предыдущих).
- `k = n`: только двуглавые → ответ `1`.

## Программа

- `src/main.cpp` — C++17.

### Аргументы

```text
./task_09 n k [trials]
```

- `n` — число монет;
- `k` — число двуглавых монет (`0 <= k <= n`);
- `trials` — число экспериментов Монте-Карло (по умолчанию `1000000`).

В каждом эксперименте: равновероятно выбирается монета, делается 4 броска. Учитываются только те эксперименты, где первые три результата — орёл; среди них считается доля, где четвёртый тоже орёл.
