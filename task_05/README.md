# task_05 - Случайное блуждание на оси Ox (JSON-конфиг)

## Постановка

Материальная точка стартует из `0`.

На каждом шаге:
- с вероятностью `p` смещается вправо на `s_plus`;
- с вероятностью `q = 1 - p` смещается влево на `s_minus`.

Для заданных значений `N` нужно эмпирически оценить вероятность события:

- `X_N = 0` (после `N` шагов точка находится в начале координат).

Параметры должны задаваться через JSON-файл.

## Что сделано

- `src/main.cpp` - программа на C++17;
- `config.json` - пример конфигурации.

Программа:
- читает `p, s_plus, s_minus, trials, target_N` из JSON;
- выполняет `trials` независимых экспериментов;
- для каждого `N` из `target_N` считает `P_hat(X_N = 0)`.

## Формат JSON

```json
{
  "p": 0.5,
  "s_plus": 1,
  "s_minus": 1,
  "trials": 200000,
  "target_N": [2, 4, 6, 8, 10, 20]
}
```

Ограничения:
- `0 <= p <= 1`;
- `s_plus > 0`, `s_minus > 0`;
- `trials >= 1`;
- каждый `N` в `target_N`: `1 <= N <= 2^32`.

## Запуск в Ubuntu (WSL)

### 1) Открыть Ubuntu

Из PowerShell:

```powershell
wsl -d Ubuntu
```

### 2) Перейти в папку задания

```bash
cd /mnt/c/CursWorkTV/task_05
```

### 3) Скомпилировать

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic src/main.cpp -o task_05
```

### 4) Запуск с конфигом по умолчанию

```bash
./task_05
```

### 5) Запуск с другим конфигом

```bash
./task_05 my_config.json
```
