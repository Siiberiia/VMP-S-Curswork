# task_01 - Проверка теоремы сложения вероятностей на перестановках

## Постановка

Дано множество из `N` различных элементов `{1, 2, ..., N}`.  
Рассматриваются все равновероятные перестановки.

Событие `A_i`: элемент с номером `i` находится на позиции `i`.

Нужно проверить (перебором и аналитически):

`P(A_i U A_j) = P(A_i) + P(A_j) - P(A_i ∩ A_j)`.

Также нужно вывести все благоприятствующие исходы для:
- `A_i U A_j`
- `A_i ∩ A_j`

## Структура

- `src/main.cpp` - программа на C++17.

## Аргументы программы

```text
./task_01 N [i j]
```

- `N` - количество элементов (`N >= 1`) - обязательный.
- `i`, `j` - индексы событий `A_i`, `A_j` - необязательные.
  - если не заданы:
    - при `N = 1`: `i = 1`, `j = 1`
    - при `N >= 2`: `i = 1`, `j = 2`

## Аналитическая часть (для отчета)

Для равновероятных перестановок:

- Общее число исходов: `N!`.
- `A_i`: фиксируем `i` на позиции `i`, остальные `N-1` элементов переставляются свободно:
  - `|A_i| = (N-1)!`, значит `P(A_i) = (N-1)! / N! = 1/N`.
- Аналогично `P(A_j) = 1/N`.
- Если `i != j`, то для `A_i ∩ A_j` фиксируются две позиции:
  - `|A_i ∩ A_j| = (N-2)!`,
  - `P(A_i ∩ A_j) = (N-2)! / N! = 1 / (N(N-1))`.
- Тогда:
  - `P(A_i U A_j) = P(A_i) + P(A_j) - P(A_i ∩ A_j)`.

Программа печатает вероятности:
- полученные прямым перебором;
- полученные по аналитическим формулам;
- модуль разности левой и правой части теоремы.
