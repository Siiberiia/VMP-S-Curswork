# task_10 — Семьи и дети (Пуассон + условие «нет девочек»)

## Условие

Число детей в семье `K` имеет распределение Пуассона:

`P(K = k) = e^{-λ} λ^k / k!`, `λ = 1.94`, `k = 0, 1, 2, …`

При фиксированном `k` все `2^k` комбинаций полов равновероятны (у каждого ребёнка независимо `P(мальчик) = P(девочка) = 1/2`).

Нужно найти **условную** вероятность `P(K ≥ m | нет девочек)` (то есть все дети — мальчики), сравнить с **безусловной** `P(K ≥ m)` и оценить симуляцией.

В формулировке в методичке часто пишут `1 ≤ m < k` — имеется в виду фиксированный порог `m ≥ 1` (а `k` — переменная числа детей).

## Аналитика (для отчёта)

Событие «нет девочек» = «все мальчики» при данном `k` имеет вероятность `(1/2)^k`.

`P(K = k, нет девочек) = P(K=k) * 2^{-k} = e^{-λ} (λ/2)^k / k!`.

Суммируя по `k`:

`P(нет девочек) = e^{-λ} e^{λ/2} = e^{-λ/2}`.

Числитель для `K ≥ m`:

`P(K ≥ m, нет девочек) = e^{-λ} ∑_{k=m}^∞ (λ/2)^k / k!`.

Условная вероятность:

`P(K ≥ m | нет девочек) = e^{-λ/2} ∑_{k=m}^∞ (λ/2)^k / k! = P(Y ≥ m)`,

где `Y ~ Poisson(λ/2)`.

Безусловно:

`P(K ≥ m) = P(Z ≥ m)`, где `Z ~ Poisson(λ)`.

Так как `λ/2 < λ`, при `m ≥ 1` обычно **`P(K ≥ m | нет девочек) < P(K ≥ m)`** — при информации «все мальчики» распределение числа детей сдвигается к меньшим значениям (много мальчиков при большом `k` маловероятно).
