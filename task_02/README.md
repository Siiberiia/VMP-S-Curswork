# task_02 - Эксперимент с математической монетой

## Идея задания

Монету бросают до первого появления двух одинаковых результатов подряд.  
Пусть `T` - номер броска, на котором эксперимент остановился.

Пространство элементарных исходов:

`Omega = {HH, TT, HTT, THH, HTHH, THTT, ...}`

Каждому исходу длины `n` соответствует вероятность `1 / 2^n`.

## Что нужно найти

1. `A`: эксперимент закончится **до** `k`-го броска (`T < k`).
2. `B`: для завершения эксперимента потребуется чётное число бросаний (`T` - чётное).

И сравнить:
- аналитические вероятности;
- эмпирические вероятности (моделирование Монте-Карло).

## Формулы (аналитика)

Для `n >= 2`:

- `P(T = n) = 1 / 2^(n-1)`.

Тогда:

- `P(T < k) = sum_{n=2}^{k-1} 1/2^(n-1) = 1 - 1/2^(k-2)` при `k > 2`, иначе `0`.
- `P(T чётное) = sum_{m=1}^{inf} 1/2^(2m-1) = 2/3`.
